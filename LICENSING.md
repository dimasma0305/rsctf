RSCTF is free and open-source software distributed under the
[MIT License](LICENSE.txt).

The MIT License applies to the original RSCTF source code in this repository.
Files that carry their own license notice remain available under those terms.

## Separately licensed third-party code

RSCTF vendors a source subset of CreepJS under the MIT License. Its license is
included beside the source at
[`web/src/lib/creepjs/LICENSE`](web/src/lib/creepjs/LICENSE).

## Copies included in the runtime image

The production container exposes these files below `/app/web/build/legal/`:

- `LICENSING.md`
- `LICENSE.txt`
- `NOTICE`
- `third-party/CreepJS-LICENSE.txt`
