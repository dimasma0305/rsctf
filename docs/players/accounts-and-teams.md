# Accounts and teams

Your account identifies you; a team is the group that joins and scores in a game.

## Create and secure your account

Use **Register** on the home page. Depending on the event, your account may need email confirmation or administrator approval before it becomes active.

After signing in, open your account menu to update your profile, avatar, email, or password. Password recovery only works when the operator has configured outgoing email.

::: tip Use a password manager
A CTF account can expose private team and event information. Use a unique password and do not share one account between teammates.
:::

## Create a team

Open **Teams** and select the create action. The creator becomes the captain. A captain can:

- Change the team profile and avatar
- Regenerate the invite code
- Remove members
- Transfer captaincy
- Delete the team when event locks allow it

An account can own or belong to a limited number of teams. rsctf currently caps the total at three teams per user and 100 members per team, although each game can set a much smaller team-size limit.

## Join an existing team

Ask the captain for the current invite code, open **Teams**, and use the join/accept flow. Treat the invite code like a password: anyone with it may be able to request or obtain membership.

## Join a game

Open **Games**, choose an event, and select **Join**. Then:

1. Choose one of your teams.
2. Choose a division if the event uses divisions.
3. Enter the game invite code if one is required.
4. Submit the request.

Your participation is either accepted immediately or marked **Pending** for organizer review. Once accepted, the game can lock the team roster. This protects competitive integrity, but it also means the captain may be unable to add, remove, or transfer members until the lock is released.

## Before the game starts

- Confirm every intended teammate is present before joining a roster-locking game.
- Read the event rules and notices.
- Test your browser login from the device you will use.
- For A&D, install WireGuard and prepare the SSH public key you intend to use.
